Description
This landing page is for promoting the app. MyLGU is a government service website that provides citizens with easy access to local services, announcements, and community resources. 

Features
Centralized access to local government services
Public announcements and updates
Job listings and community information
Clean and user-friendly interface
Responsive design for desktop and mobile

Technologies Used
HTML
CSS / Tailwind CSS
JavaScript

Purpose
This project aims to improve communication between local government units and citizens by making public services and information more accessible online.
